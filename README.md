# React App boilerplate

Sample boilerplate with react 17, webpack 5, babel, hmr, jest, eslint, prettier

### How to use

```
  git clone git@github.com:seballos/react-app-boilerplate.git
  cd react-app-boilerplate
  # run dev server
  npm start
  # build bundle
  npm build
```
